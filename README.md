# NewsAPI

## How to start the Web Service locally
- clone the repository
  - `git clone https://github.com/ivonu/news_api.git`
- bavigate to cloned repository
  - `cd news_api`
- start application with docker compose
  - `docker compose up`
- open http://localhost:8080/swagger
